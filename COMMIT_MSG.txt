feat: Initial implementation of cfctl - Cloudflare CLI Management Tool

Complete implementation of a production-ready CLI tool for managing Cloudflare
cache operations with interactive TUI and secure credential storage.

## Core Application (cmd/)

cmd/cfctl/main.go: Application entry point with Cobra CLI framework
- Initializes configuration on startup
- Launches Bubble Tea interactive TUI
- Handles graceful errors and exit codes

## API Integration (internal/api/)

internal/api/client.go: Cloudflare API client wrapper using cloudflare-go v0 SDK
- Supports both API token and global API key authentication
- Implements timeout and context management
- Provides credential verification via VerifyAPIToken()

internal/api/zones.go: Zone/domain operations
- ListZones(): Fetches all zones for authenticated account
- GetZone(): Retrieves specific zone by ID
- Maps Cloudflare API responses to internal types

internal/api/cache.go: Cache purge operations
- PurgeCache(): Unified method supporting 5 purge types:
  * Purge by URL (single/multiple files)
  * Purge by Hostname (entire domains)
  * Purge by Tag (Enterprise feature)
  * Purge by Prefix (URL prefixes)
  * Purge Everything (full zone cache)
- Builds appropriate PurgeCacheRequest based on input parameters

## Configuration Management (internal/config/)

internal/config/config.go: Configuration file management with Viper
- Load(): Auto-creates config at ~/.config/cfctl/config.yaml
- Save(): Persists configuration changes
- AddAccount()/RemoveAccount(): Account management
- GetDefaultAccount()/SetDefaultAccount(): Default account handling
- Supports environment variable CFCTL_CONFIG for custom paths

internal/config/validator.go: Input validation functions
- ValidateEmail(): RFC-compliant email validation
- ValidateAccountName(): Alphanumeric + dash/underscore, max 50 chars
- ValidateAPIToken(): Minimum 40 characters
- ValidateAPIKey(): Minimum 32 characters

internal/config/accounts.go: Secure credential storage via system keyring
- StoreCredential(): Saves to macOS Keychain/Linux Secret Service
- GetCredential(): Retrieves from keyring
- DeleteCredential(): Removes from keyring
- Service name: "cfctl" for keyring identification

internal/config/validator_test.go: Comprehensive unit tests (12 test cases)
- Email validation tests (valid/invalid formats)
- Account name validation (length/characters)
- API token validation (length requirements)

## User Interface (internal/ui/)

internal/ui/styles.go: Lip Gloss styling definitions
- Color scheme: Cyan primary, orange secondary, semantic colors
- Style constants for titles, menus, errors, borders, inputs

internal/ui/welcome.go: Welcome screen with ASCII art logo
- Displays cfctl version
- Shows account count
- Transitions to main menu on Enter

internal/ui/menu.go: Main navigation menu
- 6 options: Configure, Select Account, Domains, Settings, Help, Exit
- MessageModel for displaying alerts and notifications
- Keyboard navigation with arrow keys

internal/ui/account_config.go: Interactive account configuration flow
- Multi-step wizard: Auth type → Credentials → Verify → Success
- Text inputs for account name, email, API token/key
- Async credential verification with loading state
- Automatic keyring storage on success

internal/ui/account_select.go: Account switching interface
- Filterable list of configured accounts
- Sets default account via config.SetDefaultAccount()
- Shows currently selected account

internal/ui/domain_list.go: Domain/zone listing with filtering
- Async zone loading from Cloudflare API
- Displays zone name, status, and plan type
- Navigates to purge menu on zone selection
- Searchable with real-time filtering

internal/ui/purge_menu.go: Cache purge method selection
- 6 options: URL, Hostname, Tag, Prefix, Everything, Back
- Routes to specific purge operation screens
- Context-aware navigation

internal/ui/purge_operations.go: URL and Hostname purge implementations
- PurgeByURLModel: Textarea for comma-separated URLs
- PurgeByHostnameModel: Textarea for comma-separated hostnames
- Input validation using utils.ValidateURLs/ValidateHostnames
- Async purge execution with success/error states

internal/ui/purge_advanced.go: Tag, Prefix, and Everything purge
- PurgeByTagModel: Enterprise feature with warning message
- PurgeByPrefixModel: URL prefix purging
- PurgeEverythingModel: Full cache purge with double confirmation
- Safety checks and user confirmations

internal/ui/settings.go: Configuration display screen
- Read-only view of current config.yaml values
- Shows API settings, UI preferences, cache settings
- Account list display

internal/ui/help.go: Comprehensive help documentation
- Keyboard shortcuts reference
- Feature explanations
- Links to Cloudflare API documentation
- Troubleshooting tips
